# ngui-benchmark

A benchmarking project for comparing the performance of [Red Hat Design System (RHDS)](https://ux.redhat.com) native Web Components against PatternFly 6 (PF6) React-based components wrapped in a Web Component. PF6 components are React-only by design; wrapping them in a Web Component makes them consumable outside of React — in other frameworks, platforms, or framework-free environments. The goal is to measure and understand the real-world performance trade-offs of that approach versus purpose-built framework-agnostic Web Components.
